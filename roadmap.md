# MVP Roadmap

## Phase 1

- Create project structure
- Add DB models
- Build ingestion for Yahoo RSS
- Store articles and dedupe

## Phase 2

- Add OpenAI summarization
- Generate short podcast scripts
- Save summaries to the database

## Phase 3

- Add TTS and mp3 generation
- Store audio files locally or in object storage

## Phase 4

- Build FastAPI endpoints
- Add a simple UI
- Show latest episode and source links

## Phase 5

- Add logging, retries, and monitoring
- Add prompt evaluation
- Improve ranking and audio quality
